# waraga-2024-2025
Личный проект. Был написан на WPF зимой 2024-2025 примерно за неделю, как и остальные "личные проекты".
Отличительная черта: попытка в псевдо реализации трёхмерной графики без лучей. Возможность создания собственных карт и перемещения между ними с помощью специальной кодировки.
Были реализованы только три объекта: поверность, по которой можно передвигаться, стена, блокирующая обзор и передвижение и дверь, позволяющая перейти на следующую локацию.

Управление: 
- передвижение - WASD
- поворот камеры - QE
- открыть/закрыть миникарту - M
- выйти в меню - Esc

Через нажатие на "Завершить игру" в меню, можно перейти в специальный редактор карт, который скорее был площадкой для экспериментов. Сохранение карт завершено не было из-за истечения времени, однако порисовать возможность имеется.
